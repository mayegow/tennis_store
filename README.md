# About
This project run on ReactJs v. Hydrogen on VITE framework
TENNIS_STORE contains:
* Components
* React hydrogen
* Shopify
* Bootstrap
---
# Dependencies

* Bootstrap
* Shopify/cli
* Hydrogen

# How to run

    $ git clone <this_repo>
    $ cd tennis_Store
    $ npm i 
    $ npm run dev
# If project not run

* Create .env File on root of project
* And write on this : SESSION_SECRET=your-secret-key
* Run project
